### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/iprime2/krishil-social
```

### Install packages

```shell
yarn 
```

### Start the app

```shell
yarn run start
```

## Available commands

Running commands with yarn `yarn run [command]`
